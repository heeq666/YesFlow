## 📌 概述

YesFlow 是一个**可视化 AI 任务编排工具**，通过直观的拖拽编辑器，将多个 AI 节点连接成完整的工作流。配置条件分支、并行处理，执行结果一目了然——**全程无需写代码**。

---

## ✨ 功能特性

- **🎨 可视化流程编辑器** — 拖拽连接 AI 节点，构建工作流像画流程图一样简单
- **🤖 多模型支持** — 为不同节点配置不同的 AI 模型
- **🔄 条件与分支** — Condition 节点创建智能分支逻辑
- **🔀 并行执行** — 同时运行多个 AI 任务
- **🖱️ 智能多选** — `Shift + 点击` / `Shift + 框选`
- **🎹 完全自定义快捷键** — 为每个操作配置你喜欢的快捷键
- **🌙 深色模式** — 精心设计的深色界面
- **💾 本地优先存储** — 所有数据保存在浏览器 localStorage
- **📦 一键部署** — 纯前端，Netlify/Vercel 秒级上线

## 🚀 快速开始

```bash
git clone https://github.com/heeq666/YesFlow.git
cd YesFlow
npm install
npm run dev
```

首次使用需在**设置 → 关于系统**中填入 MiniMax API Key。
